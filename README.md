# ConversorDec2bin

##Como funciona el código del archivo ConversorDec2Bin.py
	1. Insertar un número decimal positivo.
	2. Insertar un número de bits.
	3. Convierte un número decimal positivo a binario usando el determinado número de bits que insertamos anteriormente.
	4. En la pantalla se puede observar el resultado obtenido.
